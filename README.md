#  Quiz Application

A Trivia App that asks questions with True/False answers and, dynamic buttons to advise where questions are answered correctly or incorrectly.

This app was built as part of 'iOS & Swift - The complete iOS App Development Bootcamp' by Dr Angela Yu

# Required Skills

* Use of the Model-View-Controller (MVC pattern)
* Swift Structures & instantiating a struct instance.
* Use of value & reference types. 
* Swift Classes & Object Creation.
* Use of Classes & Structs
* Code Refactoring
